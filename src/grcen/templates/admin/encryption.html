Key Management

The encryption key is provided via the ENCRYPTION_KEY environment variable (base64url-encoded, 32 bytes).

Generate a key

grcen generate-key

Rotate keys

  1. Set ENCRYPTION_KEY to the new key.
  2. Set ENCRYPTION_KEY_RETIRED to the old key.
  3. Restart the application.
  4. Run grcen rotate-keys to re-encrypt all data with the new key.
  5. Remove ENCRYPTION_KEY_RETIRED and restart.
